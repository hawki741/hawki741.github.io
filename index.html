GitHub Pages로 HTML 배포하기

  1. gitgub.com에 접속하여 github 계정 만들기
  2. 새 repository 만들기
    • Repository name은 사용자ID.github.io로 생성해야 함 (사용자ID를 정확하게 써줘야하는 것이 GitHub Page를 생성하는 규칙이므로 스펠링도 틀리면 안됨)
    • "Add a README file"을 체크
    • github에서 사용자ID/사용자ID.github.io repository 확인
  3. 터미널에 local 폴더로 복사하기(local 컴퓨터에 git 설치되어 있어야 함)
    • git clone https://github.com/사용자ID/사용자ID.github.io
    • local 컴퓨터에(git clone을 실행한 위치) 사용자ID.github.io 폴더 생성, readme.md 파일만 포함
  4. local 폴더에 index.html 생성하고 간단한 html 문서 작성하기
  5. local 폴더의 문서를 github로 push
    • git add index.html 또는 git add .
    • git commit -m "Add index.html"
    • git push
      • Username은 github의 사용자ID
      • Password는 gitgub의 personal access code (Settings - Developer settings - Personal access tokens - Generate new token
